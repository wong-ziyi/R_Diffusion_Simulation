Solution of 2nd Order Partial Differential Equations: Diffusion Equation

The objective of this section is solve diffusion equation problems using numerical methods. The diffusion equation for temperature scalar field (\(T\)) is given by

$$ \rho c_p \frac{\partial T}{\partial t} = K \frac{\partial^2 T}{\partial x^2} + \dot{q} $$

where \(\rho\) is density, \(c\) is specific heat, and \(K\) is thermal conductivity. The term \(\dot{q}\) represents energy generation term. The intial and boundary conditions needed to solve this equation can be Dirichlet (initial and boundary value of \(T\) is known) or Neumann (initial and boundary value of \(\tfrac{dT}{d\eta}\) is known) or mixed. This equation can solved using finite difference method giving

$$ \rho c_p \frac{T^{(n+1)}_i - T^{(n)}_i}{\Delta t} = K \frac{T^{(n)}_{i+1} - 2T^{(n)}_{i} + T^{(n)}_{i-1}}{\Delta x^2} + \dot{q}_i^{(n)} $$

where \(n\) and \(i\) are time and x-axis indices. We can rearrange this equation to get

$$ T^{(n+1)}_i = T^{(n)}_i + \frac{\Delta t}{\rho c_p} \bigg[ K\frac{T^{(n)}_{i+1} - 2T^{(n)}_{i} + T^{(n)}_{i-1}}{\Delta x^2} + \dot{q}_i^{(n)} \bigg] $$

1. One-Dimension with Dirichlet and Neumann Coundary Conditions

Consider a 10-m long thin metal rod at a temperature of 100oC. How would the temperature of the rod evolve over time if

  1. at \(t = 0\) both of its ends are placed on ice (0oC)?
  2. at \(t = 0\) right end is placed on ice (0oC)?

Assume that the heat can only flow between rod and ice.

With not source (\(\dot{q}^{(n)}_i = 0\)), this is gives

$$ T^{(n+1)}_i = T^{(n)}_i + \Delta t \frac{K}{\rho c_p} \bigg[ \frac{T^{(n)}_{i+1} - 2T^{(n)}_{i} + T^{(n)}_{i-1}}{\Delta x^2} \bigg] $$

with auxiliary conditions for two parts as follow

  1. Dirichlet

    2. \( T(x,0)=100 \hspace{ 87px } 0 \leq x \leq 10 \hspace{ 15px } \) Initial Condition

    \( T(0,t)=T(10,t)=0 \hspace{ 30px} t \gt 0 \hspace{ 59px } \) Boundary Conditions

  2. Neumann

    3. \( T(x,0)=100 \hspace{ 20px } 0 \leq x \leq 10 \hspace{ 15px } \) Initial Condition

    \( \left. \tfrac{ \partial T }{ \partial x } \right|_{ x = 0 } = 0 \hspace{ 35px} t \gt 0 \hspace{ 59px } \) Boundary Conditions

    \( \left. \frac{ \partial T }{ \partial x } \right|_{ x = 0 } \approx \frac{T^{(n)}_1 - T^{(n)}_0}{\Delta x} = 0 \hspace{ 53px} \) which implies \( T^{(n)}_1 = T^{(n)}_0 \).

2. One-Dimension with Source

A long time has passed and now the temperature of the rod has become 0oC. Suppose we reset the timer to \(t=0\) and bring a candle flame and place it under the rod 7-m from the left end for 5 seconds. If we treat the flame as energy generation of 10 units in the rod, how would the temperature of the rod evolve over time for parts (1) and (2) of the last problem?

In this case, we will use the equation

$$ T^{(n+1)}_i = T^{(n)}_i + \Delta t \frac{K}{\rho c_p} \bigg[ \frac{T^{(n)}_{i+1} - 2T^{(n)}_{i} + T^{(n)}_{i-1}}{\Delta x^2} \bigg] + \Delta t \frac{\dot{q}_{n,i}}{\rho c_p} $$

where the boundary conditions are the same as before. Clearly, there is a source component that needs to be considered. The source function can be written as

$$ \begin{eqnarray} \dot{q}(t,7)= \begin{cases} 10 \hspace{ 15px } 0 \leq t \leq 5 \\ 0 \hspace{ 23px } t \gt 5 \end{cases} \end{eqnarray} $$
